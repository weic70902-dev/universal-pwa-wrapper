# PWA Wrapper

这是一个用于包装无法嵌入的网站的PWA壳应用。适用于目标网站禁止被 iframe 嵌入（如 Google、GitHub 等）的情况。

## 工作原理

- 作为独立的PWA应用安装在设备上
- 打开应用时可以输入任意网站URL
- 点击跳转按钮即可访问指定的网站
- 绕过目标网站的X-Frame-Options限制

## 功能特点

- 支持自定义输入任意网站URL
- 支持保存常用的网站URL
- 支持通过回车键快速跳转
- 支持安装到主屏幕，像原生应用一样使用

## 使用方法

1. 克隆此仓库或直接访问部署地址
2. 在输入框中输入您要访问的网站完整URL（包含https://）
3. 点击"跳转到网站"按钮访问该网站
4. （可选）点击"保存URL"按钮保存当前URL，方便下次使用
5. （可选）将应用安装到桌面，获得类似原生应用的体验

## 安装说明

### 浏览器安装步骤
1. 在支持PWA的浏览器中打开此应用
2. 点击地址栏旁边的安装按钮或菜单中的"安装"选项
3. 确认安装信息
4. 应用将安装到您的桌面或主屏幕

### 支持的浏览器
- Chrome（桌面版和移动版）
- Edge（桌面版和移动版）
- Firefox（移动版）
- Safari（iOS 11.3+）

## 文件说明

- `index.html`: 主应用程序文件，包含用户界面和跳转逻辑
- `manifest.json`: PWA配置文件
- `sw.js`: Service Worker文件，处理离线功能
- `icon.png`: 应用图标

## 支持的网站

理论上支持所有网站，包括：
- Google
- GitHub
- YouTube
- Twitter
- Facebook
- 等等

## 技术细节

### URL存储
应用使用localStorage来保存用户输入的URL，数据仅存储在本地浏览器中，不会上传到任何服务器。

### Service Worker
Service Worker采用网络优先策略，确保始终获取最新的网络资源。

## 注意事项

1. 目标网站必须可以通过HTTPS访问
2. 某些网站可能有额外的安全限制
3. 用户需要在支持PWA的浏览器中使用此应用
4. 保存的URL数据仅在当前浏览器中可用
